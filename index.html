  font-weight: 600;

  transition: all 0.3s ease;
}

  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;

  transform: translateY(-2px);
  box-shadow: 0 8px 25px rgba(102, 126, 234, 0.4);
}

Documentation

Learn how to use and customize these components

Button Component

Interactive buttons with hover effects and multiple variants. Supports primary, secondary, and outline styles.

  • Classes: .demo-btn, .primary, .secondary, .outline
  • Features: Hover animations, ripple effect, gradient backgrounds
  • Usage: Add the base .demo-btn class and a variant class (e.g., primary).

Card Component

Flexible cards with image and text content. Perfect for displaying products, articles, or profiles.

  • Structure: .demo-card > .card-image + .card-content
  • Features: Glass morphism effect, hover animations
  • Responsive: Adapts to different screen sizes

Form Component

Styled form inputs with focus states and validation. Includes input fields and submit buttons.

  • Elements: .demo-form, .form-group, .form-input, .form-submit
  • Features: Floating labels, focus animations, modern styling
  • Usage: Wrap inputs in a .form-group inside a .demo-form.
